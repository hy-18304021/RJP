
index{
DBからスレッド名と作成日時、更新時間を受け取りたい
スレッド名の検索結果をDBから受け取りたい
ログインしているならログインの上にIDを表示してアカウント製作ボタンにはログアウトボタン
}

threadcreate{
スレッド製作するさいに件名が入っているか判定
}

res{
ユーザー名と登校時間、レスの内容をDBから受け取りたい
スレッド名をDBから受け取りたい
ログインしているならログインの上にIDを表示してアカウント製作ボタンにはログアウトボタン
}

res{
レスを投稿する際に文が入っているかを判定
}

usercreate{
ユーザー名とパスワードをDBに登録したい
パスワードが一致しているか判定
}

userlogin{
ユーザー名とパスワードがDBにあり完全一致しているか判定
}
